# Spring Legacy Project | Spring MVC Project.

Proyecto correspondiente al Módulo 6 del Bootcamp Java Full Stack con FUDESCO.

Este proyecto abarca el contenido correspondiente a las actividades grupales 1, 2, 2.1,
3, 3.1, 4, 4.1, 5, 5.1, 7, 7.1, 8 y 9.

El proyecto se encuentra versionado con Git, en la historia de commits se puede apreciar 
los puntos en los que cada grupal fue desarrollado e implementado en esta aplicación.

Durante el desarrollo del proyecto se implementaron varias versiones de java, hasta llegar 
a la más óptima en compatibilidad con todas las dependencias del proyecto.
